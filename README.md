# 📥 Fast Save Bot

Универсальный Telegram-бот на Python для скачивания медиаконтента из популярных соцсетей: **YouTube**, **TikTok** и **Instagram**.

## 🚀 Основные возможности
* **Автоматическое распознавание ссылок**: Бот сам определяет платформу (YouTube, Shorts, TikTok, Instagram).
* **Выбор формата**: Возможность скачать видео с выбором качества или только аудиодорожку (MP3).
* **Умная обработка**: Автоматическая склейка видео и аудио потоков через FFmpeg для получения лучшего качества.
* **Docker-ready**: Проект полностью готов к деплою на сервер через Docker контейнеры.

## 🛠 Технологический стек
* **Язык:** [Python 3.11+](https://www.python.org/)
* **Библиотека бота:** [aiogram 3.x](https://docs.aiogram.dev/) (Asynchronous)
* **Движок загрузки:** [yt-dlp](https://github.com/yt-dlp/yt-dlp)
* **Обработка медиа:** [FFmpeg](https://ffmpeg.org/)
* **Контейнеризация:** [Docker](https://www.docker.com/) & [Docker Compose](https://docs.docker.com/compose/)

## 📂 Структура проекта
```text
fast_save_bot/
├── bot.py                # Точка входа и запуск бота
├── config.py             # Настройки (чтение .env)
├── core/                 # Основная логика загрузки (yt-dlp)
├── handlers/             # Обработчики сообщений и нажатий кнопок
├── keyboards/            # Генерация инлайн-кнопок выбора качества
├── downloads/            # Временная папка для медиафайлов
├── Dockerfile            # Инструкции для сборки образа
└── docker-compose.yml    # Конфигурация запуска контейнера

⚙️ Установка и запуск (Локально)
1. Клонируйте репозиторий:
Bash:
git clone https://github.com/bek07kg/fast_save_bot.git
cd fast_save_bot

2. Создайте файл .env в корне проекта и добавьте ваш токен:
Фрагмент кода: 
BOT_TOKEN=ВАШ_ТЕЛЕГРАМ_ТОКЕН

3. Установите зависимости:
Bash:
pip install -r requirements.txt
Примечание: убедитесь, что FFmpeg установлен в вашей системе и добавлен в PATH.

4. Запустите бота:
Bash:
python bot.py

🐳 Запуск через Docker (Для сервера)
Это рекомендуемый способ для деплоя на ваш сервер (старый ноутбук). Docker автоматически установит нужную версию Python и FFmpeg внутри контейнера.
Bash:
# Сборка и запуск в фоновом режиме
docker-compose up -d --build

⚠️ Ограничения
Размер файлов: По умолчанию лимит Telegram Bot API составляет 50 МБ. Файлы больше этого размера бот отправить не сможет до настройки локального Bot API сервера.

TikTok в заблокированных регионах: Если сервер находится в регионе, где TikTok заблокирован, потребуется настройка прокси/VPN внутри Docker-контейнера.
